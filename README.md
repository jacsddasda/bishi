# 智能简历分析系统

## 项目概述

智能简历分析系统是一个基于AI的招聘辅助工具，能够自动解析PDF格式的简历，提取关键信息，并与岗位需求进行匹配评分，帮助招聘者快速筛选候选人。

## 系统架构

### 后端服务
- **技术栈**：Python + Flask
- **核心功能**：
  - PDF简历上传和解析
  - 关键信息提取（基本信息、求职信息、背景信息）
  - 简历与岗位需求匹配评分
  - 结果缓存（Redis）

### 前端页面
- **技术栈**：HTML + CSS + JavaScript
- **核心功能**：
  - 简历文件上传
  - 解析结果展示
  - 岗位需求输入
  - 匹配度评分展示

## 快速开始

### 后端服务
1. 安装依赖：
   ```bash
   cd backend
   pip install -r requirements.txt
   ```

2. 启动服务：
   ```bash
   python app.py
   ```
   服务将在 http://localhost:5000 运行

### 前端页面
1. 直接打开 `frontend/index.html` 文件
2. 或部署到任何静态文件服务器

## API接口

### 1. 上传简历
- **URL**：`/api/upload`
- **方法**：POST
- **参数**：`file` (PDF文件)
- **返回**：解析后的简历信息

### 2. 计算匹配度
- **URL**：`/api/score`
- **方法**：POST
- **参数**：
  - `resume_info`：解析后的简历信息
  - `job_description`：岗位需求描述
- **返回**：匹配度评分结果

## 功能特性

1. **PDF简历解析**：支持多页PDF文件的解析和文本提取
2. **关键信息提取**：自动提取姓名、电话、邮箱、地址、求职意向、期望薪资、工作经验、教育背景和项目经历
3. **智能匹配评分**：基于技能、经验和教育背景的综合评分
4. **结果缓存**：使用Redis缓存已解析的简历，提高响应速度
5. **用户友好界面**：简洁直观的前端页面，支持拖拽上传和实时反馈

## 技术亮点

- **模块化设计**：清晰的代码结构，便于维护和扩展
- **AI辅助**：利用正则表达式和关键词匹配实现智能信息提取
- **高性能**：PDF解析和信息提取速度快，支持多页简历
- **可扩展性**：预留了AI模型集成接口，可进一步提升分析能力

## 部署建议

- **生产环境**：可部署到阿里云函数计算FC或其他Serverless平台
- **存储**：建议使用对象存储服务存储上传的简历文件
- **缓存**：生产环境建议使用Redis集群提高缓存性能
- **安全**：添加文件类型验证和大小限制，防止恶意上传
